# Self-made-oj
开源免费的判题系统，有题库。整体编程语言只有Python 和 html
# 关于题库
题库里面有经典的编程题————两数之和，还有在线GESP2023年03月的试卷
- # 咋运行？
首先，你的电脑要安装Python3.8以上的Python版本（必须）<br>
其次，GCC/JAVA/NODEJS请自行安装（可选，只不过如果不安装，oj里面的编程语言只有Python能用）<br>
第二，运行main.py，然后再运行oj/judge_core.py<br>
之后就可以打开浏览器，访问127.0.0.1:1279查看oj啦，也可以访问127.0.0.1:8080查看判题接口。
# 咋加题目？
请自觉复制粘贴，改题目
